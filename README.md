# 💖 Memory Card Game 🎮

## 📌 Description
This is a simple and interactive Memory Card Game developed using HTML, CSS, and JavaScript. The game tests the player's memory by matching pairs of cards.

## 🎯 Features
- 🧠 Flip cards to find matching pairs
- ⏱ Timer to track time
- 🎯 Move counter
- 🔄 Restart button
- 🎉 Winning alert

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript

## 🚀 How to Play
1. Click on any card to flip it
2. Click another card to find a match
3. If both match → they stay open
4. If not → they flip back
5. Match all pairs to win
